# Sueño Andino - Sitio Web

Sitio web profesional para Sueño Andino, organización dedicada al desarrollo territorial regenerativo en comunidades andinas de Latinoamérica.

## 📁 Estructura del Proyecto

```
SueñoAndino/
├── sueño-andino-wordpress.html    # Página principal del sitio
├── blog-sueño-andino.html         # Página de blog con artículos
└── README.md                      # Este archivo
```

## 🚀 Características

### Página Principal (`sueño-andino-wordpress.html`)
- **Diseño moderno y profesional** con gradientes y efectos visuales
- **Navegación funcional** con smooth scrolling entre secciones
- **Secciones principales:**
  - Hero con estadísticas impactantes
  - Golden Circle (Por qué existimos, Qué hacemos, Cómo lo hacemos)
  - Timeline de la historia de la organización
  - Servicios con iconos y descripciones
  - Testimonios de comunidades
  - Equipo (Directorio y Equipo)
  - Blog con artículos destacados
  - Formulario de contacto funcional
- **Popup de captura de leads** para descarga de guía gratuita
- **Diseño responsive** optimizado para todos los dispositivos

### Página de Blog (`blog-sueño-andino.html`)
- **Diseño elegante** con tarjetas de artículos
- **3 artículos completos:**
  - "5 Principios de la Agricultura Regenerativa"
  - "El Poder de la Participación Comunitaria"
  - "Desarrollo Territorial en Tiempos de Cambio Climático"
- **Experiencia de lectura optimizada** con tipografía mejorada
- **Navegación fluida** entre artículos y página principal
- **Diseño responsive** para móviles y tablets

## 🎨 Diseño y Tecnologías

## 🚀 Cómo Usar

1. **Abrir la página principal:**
   ```bash
   # Abrir en navegador
   start sueño-andino-wordpress.html
   ```

2. **Navegar al blog:**
   - Hacer clic en "Blog" en el menú de navegación
   - O abrir directamente: `blog-sueño-andino.html`

3. **Probar funcionalidades:**
   - Formulario de contacto
   - Popup de guía gratuita
   - Navegación entre secciones
   - Lectura de artículos del blog
