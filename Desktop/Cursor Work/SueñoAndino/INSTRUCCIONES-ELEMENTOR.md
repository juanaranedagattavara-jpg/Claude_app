## 📋 Instalación Paso a Paso

### 1. Instalar el Tema
- Ve a **Apariencia > Temas** en WordPress
- Haz clic en **"Añadir nuevo"** > **"Subir tema"**
- Selecciona `sueno-andino-elementor-theme.zip`
- Haz clic en **"Instalar ahora"** y **"Activar"**

### 2. Instalar Elementor (si no lo tienes)
- Ve a **Plugins > Añadir nuevo**
- Busca **"Elementor"**
- Instala y activa el plugin

### 3. Configuración Automática
El tema configurará automáticamente:
- ✅ Colores globales de Elementor
- ✅ Tipografías de Elementor
- ✅ Espaciado entre widgets
- ✅ Configuración de viewports
- ✅ Soporte para todos los tipos de post

## 🔍 Troubleshooting

### **Problemas Comunes**

**Elementor no aparece:**
- Verifica que Elementor esté instalado y activado
- Ve a **Elementor > Herramientas** y regenera CSS

**Colores no se aplican:**
- Ve a **Elementor > Kit** y verifica la configuración
- Regenera CSS desde **Elementor > Herramientas**

**JavaScript no funciona:**
- Verifica que jQuery esté cargado
- Revisa la consola del navegador para errores

### **Soporte**
Para problemas específicos de Elementor:
1. Verifica la configuración del Kit
2. Regenera CSS y datos
3. Contacta al equipo de desarrollo
